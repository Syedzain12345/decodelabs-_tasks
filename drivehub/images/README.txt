This project loads car photos directly from Unsplash/Pexels CDNs
(see the <img src="..."> links in index.html), so no local image
files are needed here. If you'd rather host images locally:

1. Download each photo from the URLs used in index.html
2. Save them into this /images folder
3. Update the src="..." paths in index.html to "images/filename.jpg"
